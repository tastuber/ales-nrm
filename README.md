# ales-nrm

Data reduction pipeline for LMIRCam/ALES non-redundant aperture masking
(NRM) observations at the Large Binocular Telescope Interferometer (LBTI).

## Prerequisites

Currently, the pipeline requires reduced image cubes from the ALES pipeline,
thus fits files with the three dimensions spatial $\times$ spatial $\times$
wavelength.

## Status

The project is in early development. If you would like to get involved, please contact [the owner](mailto:tstuber@arizona.edu).
